📚 세특 탭 사용 설명서
STEP 01 / 06
➕
과목 추가하기
왼쪽 상단 과목명 입력란에 과목명을 입력하고 + 추가 버튼을 클릭하세요.
예) 체육, 수학, 국어, 영어
💡 과목은 여러 개 추가할 수 있어요. 탭으로 전환하며 관리해요!
STEP 02 / 06
🎯
성취기준 입력하기
+ 성취기준 추가 버튼으로 성취기준을 하나씩 입력하세요.
여러 개 입력 가능하며 AI가 문구 작성 시 참고해요.
💡 성취기준이 구체적일수록 더 좋은 문구가 나와요!
STEP 03 / 06
📋
활동 목록 등록하기
+ 활동 추가 버튼으로 활동명과 설명을 입력하세요.
활동설명은 선택사항이지만 입력하면 AI가 더 정확한 문구를 만들어줘요.
💡 활동명이 나중에 문구 앞에 붙으니 짧고 명확하게 써주세요!
STEP 04 / 06
🤖
AI 문구 받기
📋 AI용 텍스트 복사 버튼을 클릭한 뒤,
ChatGPT나 Claude 같은 AI 채팅창에 붙여넣으세요.
AI가 활동별·등급별 문구를 30개씩 만들어줘요.
💡 문구 형식: [활동명-A], [활동명-B], [활동명-C], [활동명-D]
STEP 05 / 06
📝
문구 뱅크에 저장하기
AI가 만들어준 문구를 문구 뱅크 텍스트창에 붙여넣고
✨ 형식 정리 버튼을 클릭하세요.
자동으로 [활동명-A] 문장 형식으로 정리돼요.
💡 형식 정리 후 전체 삭제 버튼으로 다시 시작할 수도 있어요!
STEP 06 / 06
✨
학생별 조합하기
오른쪽 패널에서 반을 선택하면 학생 목록이 나타나요.
각 학생의 등급(A/B/C/D)을 설정한 뒤
✨ 조합하기 버튼을 클릭하면 자동으로 문구가 완성돼요!
💡 결과는 수정·복사·CSV 내보내기가 모두 가능해요!
🏆 창체 탭 사용 설명서
STEP 01 / 05
📌
활동 유형 선택하기
상단 탭에서 자율활동 / 진로활동 / 동아리활동 중 작성할 유형을 선택하세요.
각 유형별로 활동과 문구를 따로 관리할 수 있어요.
💡 탭마다 독립적으로 저장돼요!
STEP 02 / 05
📅
활동 목록 등록하기
+ 활동 추가 버튼으로 날짜·활동명·설명을 입력하세요.
날짜를 선택하면 자동으로 날짜순 정렬이 돼요.
💡 활동설명에 "체육대회임" 같은 맥락을 써두면 AI가 더 잘 이해해요!
STEP 03 / 05
🤖
AI 문구 받기
📋 AI용 텍스트 복사 버튼을 클릭한 뒤
AI 채팅창에 붙여넣으세요.
활동별로 교육적 요소를 담은 문구를 15개씩 만들어줘요.
💡 문구 형식: [활동명] 문장 으로 한 줄씩 나와요.
STEP 04 / 05
📝
문구 뱅크에 저장하기
AI가 만들어준 문구를 문구 뱅크에 붙여넣고
✨ 형식 정리 버튼을 클릭하세요.
특수문자나 불필요한 기호도 자동으로 제거돼요.
💡 활동당 15개 이상 저장하면 30명도 겹치지 않아요!
STEP 05 / 05
✨
학생별 조합하기
오른쪽 패널에서 반 선택 후 활동명 형식을
활동명만 또는 활동명+날짜 중 선택하세요.
✨ 조합하기를 클릭하면 전체 활동 중 랜덤 2개가 조합돼요!
💡 날짜 포함 예시: 학교폭력예방교육(2026.03.01)에서
👥 학생관리 탭 사용 설명서
STEP 01 / 03
⚡
자동 생성으로 빠르게 등록
반 번호와 인원수를 입력하고 자동 생성 버튼을 클릭하면
학번이 자동으로 생성돼요.
학번 형식: 학년+반(2자리)+번호(2자리)
예) 1학년 1반 1번 → 10101
💡 학년은 설정 탭에서 미리 설정해주세요!
STEP 02 / 03
📋
붙여넣기로 성별 한 번에 등록
번호와 성별을 아래 형식으로 입력하고 생성 버튼을 클릭하세요.

1 남
2 여
3 남

학번은 설정한 반 번호 기준으로 자동 완성돼요.
💡 나이스 명렬표에서 복사해서 수정하면 더 빨라요!
STEP 03 / 03
✏️
개별 수정 및 반 관리
각 학생의 학번·성별은 직접 수정 가능해요.
상단 + 반 추가 버튼으로 여러 반을 만들 수 있고
탭으로 전환하며 관리해요.
결번은 삭제 버튼으로 제거하세요.
💡 설정 탭에서 JSON 백업을 해두면 데이터를 안전하게 보관할 수 있어요!
